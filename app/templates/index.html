Routes

  • GET: /api/teams - Get a list of all teams along with their total payrolls. here
  • (PAGE): /teams - Render an HTML page listing all teams.here
  • GET: /api/teams/<team_code>/players - Get a list of players for a specific team.here
  • (PAGE): /teams/<team_code>/players - Render an HTML page with a list of players for a specific team.here
  • PATCH: /api/player/<player_id>/FA - Cut a player and make them a free agent.here
  • GET: /api/free-agents - Get a list of all free agents.here
  • (PAGE): /free-agents - Render an HTML page listing all free agents.here
  • PATCH: /api/player/<player_id> - Sign a player to a team via API.here
  • (PAGE): /player/<player_id> - Render an HTML page to sign a player to a team.here
  • POST: /api/player/create - Create a new player via API.here
  • (PAGE): /player/create - Render an HTML page to create a new player.here
